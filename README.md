# Palenca app 1

## Requirements

- Node 16.5

## Common setup

Clone the repo and install the dependencies.

```bash
npm i
```

## run in development mode

To start the express server, run the following

```bash
npm run dev
```

## run in production mode

To start the express server, run the following

```bash
npm run start
```

## Curl

See postman folder in order to request app
